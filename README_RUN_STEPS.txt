IMPORTANT INTELLIJ SETUP
------------------------
1. Open the project folder in IntelliJ IDEA.
2. Wait until Maven loads.
3. Use JDK 17.
4. Go to Run > Edit Configurations.
5. Add Tomcat Server > Local.
6. Select your Tomcat 9.0.117 home folder, not TomEE.
7. Deployment > + > Artifact > TourismPackageSystem:war exploded.
8. Apply > OK > Run.
